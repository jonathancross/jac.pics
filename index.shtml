About this website

JCDSee is my personal web based photo viewing / cataloging system written in the Perl scripting language. The script is about 43KB, can be used without JavaScript and does not use cookies or contain ads. Feel free to grab the latest version of this code, and use it as you like.

This website has evolved slowly over the last decade starting from a 2-line perl script — most of the development taking place in random Internet cafes around the world. This latest version (with mobile support) was launched in September of 2015.
